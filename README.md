# Nd_Scripts
Scripts for Neon Dream

## Drug System

A comprehensive FiveM drug system with separate database tables for each drug type.

### Features
- **Growable Drugs**: Plant seeds in pots and harvest after growth period
- **Non-Growable Drugs**: Interact with AI dealers at specific locations with minigames
- **Configurable**: Easily add new drugs, modify growth times, set item drops
- **Database per Drug**: Each drug type has its own dedicated table
- **Cooldown System**: Prevents exploitation with location-based cooldowns

See [DRUG_SYSTEM.md](DRUG_SYSTEM.md) for detailed documentation.
